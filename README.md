# RetroSim

RetroSim is an open source engine recreation of Sims 1 using Sims Online. It still requires the game files for the basic usage, the main purpose is to allow users have an expanded engine with better gameplay experience and new features.

# Requirements

## Prerequisites

- .NET 5 SDK or higher
- The Sims Online
